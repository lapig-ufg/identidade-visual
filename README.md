## Diretrizes Resumidas

### Logotipo
O logotipo oficial é **intocável** — não deve ser recriado, redesenhado ou alterado. Utilizar sempre os arquivos SVG/PNG originais da pasta `01-logos/`.

### Elementos Gráficos
- **Curvas de nível** = elemento principal de **background** (texturas, fundos de slides e cards)
- **Hexágonos** = elemento **complementar** para recortes de imagens, ícones e molduras
- **Regra absoluta:** nenhuma curva de nível pode sobrepor outra

### Tipografia (Google Fonts — OFL)
- **Exo 2** — títulos e display (compatível com o wordmark do logo)
- **DM Sans** — corpo e UI
- **JetBrains Mono** — dados, código e overlines
